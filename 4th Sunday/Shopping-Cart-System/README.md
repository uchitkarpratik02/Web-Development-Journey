# Shopping Cart Management System (Console Based)

A pure console-driven prototype for inventory tracking and retail checkouts built with native Javascript.

## Structure
- Built with vanilla array manipulation sequences (`for` loops and `splice`/`push`).
- Features no DOM triggers or Array Prototype abstractions (`map`, `filter`, `reduce`).

## To Launch
1. Double click `index.html` to execute in a browser window.
2. Open DevTools dashboard (`F12` or `Cmd + Option + I` / `Ctrl + Shift + I`).
3. View runtime traces under the **Console** layer.
